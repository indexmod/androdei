T H E N E W W E B

by Andr o Dei

This lightweight one-page site is built with minimal code to make it easy to add and showcase web experiments in a personal gallery. I simply drop HTML files or links into a folder — and they appear here instantly. Based on my earlier concept Birds. A clone of this site can be used as a simple system for publicly previewing and exchanging micro-projects. If you’d like to show your support, you’re welcome to Coffee or explore my other work. Thank you.
